SWAN-Derived Wave Transfer Matrix

The GFS Wave forecast above is shown without modification. To estimate conditions at the surf spot, Sykemeter applies a SWAN-derived transfer matrix built from representative offshore wave cases near One Tree Island South.

Historic wave records were filtered for outliers and reduced to 15 representative sea states using a maximum-dissimilarity algorithm. The selected cases preserve spread in wave height, peak period and circular wave direction, with K-means averages retained as a comparison. Each MDA case was then used as offshore forcing for SWAN over the rotated local bathymetry grid. At the surf spot, grid row 75 and grid column 22, the transfer coefficient is calculated as spot Hs divided by offshore Hs. The operational 24 by 18 matrix is filled between modelled cases using periodic inverse-distance interpolation over circular direction and log-scaled peak period.
